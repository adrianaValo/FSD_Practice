Maps in JS

A Map holds Key-value Pair, where the key can be any datatype

Map always remembers the original order of insertion of keys

We can get the size of map any time

    Map has many methods
  1. new Map(): This will create new Map Object
  2. set(): sets the value for a key in a map
  3. clear(): Removes all the elements from the map
  4. has(): returns true if the key exists
  5. get(): get the value of the key in the map
  6. size(): returns the number of the elements in the map
